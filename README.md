# Multitasking — Памятка

---

## Первый запуск (только Discord-администратор сервера)

| Команда | Описание |
|---|---|
| `!роль_админ @роль` | **Первая команда.** Назначить роль, которая управляет ботом |

После этого вся настройка доступна обладателям этой роли.

---

## Настройка (только роль администратора бота)

| Команда | Описание |
|---|---|
| `!роль_взп @роль` | Роль ВЗП — тегается в `!взп` |
| `!роль_мп @роль` | Роль МП — тегается в `!взп` и `!мп` |
| `!роль_реаки @роль` | Роль для тега в `!реаки` |
| `!роль_варн 1 @роль` | Роль для варна 1/3 (аналогично 2 и 3) |
| `/тикет_менеджер @роль` | Роль тикет-менеджера — тегается в тикетах, может одобрять/отклонять |
| `/тикет #канал Категория` | Создать панель заявок в канале |
| `/лог_отказов #канал` | Канал для дублирования отказов по заявкам |
| `/магазин` | Развернуть панель магазина в текущем канале |
| `!афк` | Создать панель АФК в текущем канале |
| `!инактив` | Создать панель инактива в текущем канале |
| `/настройки` | Показать текущую конфигурацию бота (роли, каналы, панели) |

---

## Сборы (только Администратор)

| Команда | Описание |
|---|---|
| `!взп [кол-во] [название]` | Сбор ВЗП. Тегает **роль ВЗП + роль МП** |
| `!мп [кол-во] [название]` | Сбор МП. Тегает **роль МП** |
| `!реаки [кол-во] [название]` | Сбор мероприятия. Тегает **роль реаки** |

- Можно прикрепить фото к сообщению — оно появится в эмбеде
- До 24 слотов — кнопки с номерами, 25+ — одна кнопка ✅
- Создаётся тред с живым списком участников

| Команда | Где | Описание |
|---|---|---|
| `!замена <айди_кого> <айди_на_кого>` | В треде сбора | Заменить участника в слоте (0 = убрать) |

## Баллы и варны (только Администратор)

| Команда | Описание |
|---|---|
| `!дать @пользователь 500` | Начислить баллы |
| `!снять @пользователь 500` | Снять баллы |
| `!баланс [@пользователь]` | Показать баллы и варны |
| `!warn @пользователь 2 причина` | Выдать варн (1/2/3) — назначает соответствующую роль + ЛС |
| `!снять_варн @пользователь` | Снять варн и убрать варн-роль |
| `!warnlist` | Список всех участников с варнами |

## Тикеты (заявки)

**Для участников:**
- Нажать кнопку **📋 Подать заявку** в панели
- Заполнить форму: ник/статик, часы, возраст, семьи, откат
- После создания тикета можно писать в канале

**Для тикет-менеджера:**
- ✅ **Одобрить** — участнику приходит ЛС с одобрением, канал удаляется через 10 сек
- ❌ **Отклонить** — указать причину → ЛС участнику + запись в лог-канал

**Лог отказа содержит:** участника, причину, кто рассматривал

## АФК-панель

| Кнопка | Действие |
|---|---|
| 🕐 Отошел АФК | Указать причину и время возврата (например `18:30`) |
| ✅ Вернулся из АФК | Убрать себя из списка |

---

## Инактив-панель

| Кнопка | Действие |
|---|---|
| 📅 Ухожу в инактив | Указать причину и дату возврата (например `25.04.2026`) |
| ✅ Вернулся из инактива | Убрать себя из списка |

---

## Роли и доступ

| Роль | Доступ |
|---|---|
| **Администратор** | Всё — настройка, команды, тикеты |
| **Тикет-менеджер** | Только тикеты: тегается, одобряет/отклоняет |
| **Участник** | АФК-панель, инактив-панель, магазин, запись в сборы |
